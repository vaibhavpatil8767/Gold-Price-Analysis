📊 Gold Price Analysis

📌 Project Overview

This project analyzes gold price trends using historical data.
It uses data analysis and visualization techniques to understand how gold prices change over time.

---

🎯 Objectives

- Analyze historical gold price data
- Identify trends and patterns
- Compare gold with other assets (like oil, silver, etc.)
- Visualize price movements using graphs

---

🛠️ Technologies Used

- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

📂 Dataset

The dataset contains:

- Date
- Gold Price (SPX)
- Oil Price (USO)
- Silver Price (SLV)

---

📈 Features

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Visualization (graphs, trends)
- Correlation analysis

---

▶️ How to Run

1. Open the notebook in Google Colab
2. Upload dataset file
3. Run all cells

---

📸 Output

- Trend graphs 📊
- Correlation heatmaps 🔥
- Insights about gold price movements

---

🚀 Future Improvements

- Add machine learning models
- Predict future gold prices
- Deploy as a web app
